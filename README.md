#PetStore Restful API demo

A go API microservice

Prerequisite

- GO installed [https://golang.org/]
- localhost:8080 is not in use

How to run

```go get ./...```

```go run ./main.go```

Notice
-  /pet endpoint is fully operational, any others are not coded
-  Due to time limit, data is stored in memory (a hash map)
-  A sql installation file in the db folder
-  Uploaded files will be in ./uploadedFiles folder
-  Some error handling code are redundant which could be refactored
-  Curl test cases are not real automation tests
-  Models are generated by swagger auto-gen tool

TESTING

```
cd test
./curlTest.sh

```
